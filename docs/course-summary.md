# Course Summary

This repository documents hands-on work from the **Amdocs AI-Augmented Software Engineering** training program. It is organized as a portfolio: lectures build skills, homework applies them, and projects demonstrate end-to-end delivery.

## Program flow

| Stage | Location | Focus |
|-------|----------|-------|
| Lectures 01–03 | [`lectures/`](../lectures/) | Python, Jupyter, OOP, NumPy |
| Lecture 04 | [`lectures/04_nlp_rag/`](../lectures/04_nlp_rag/) | NLP, embeddings, FAISS, RAG demos |
| Lectures 05–06 | [`lectures/05_flask_intro/`](../lectures/05_flask_intro/), [`lectures/06_flask_advanced_rag/`](../lectures/06_flask_advanced_rag/) | Flask, SQLite, RAG web apps |
| Lecture 07 | [`lectures/07_docker_aws/`](../lectures/07_docker_aws/) | Docker, AWS EC2 architecture |
| Homework 01–03 | [`homework/hw01/`](../homework/hw01/) … [`hw03/`](../homework/hw03/) | Notebooks, Python exercises, CLI + tests |
| Homework 04 | [`homework/hw04/`](../homework/hw04/) | Full RAG application (in progress) |
| Homework 05 | [`homework/hw05/`](../homework/hw05/) | Ubuntu EC2, Docker, Nginx lab |
| Projects | [`projects/`](../projects/) | Capstone RAG app (IncidentIQ) |

## Skills demonstrated

- Python 3.12+, validation, testing with pytest
- Web backends: Flask and FastAPI
- Frontend: HTML/CSS/JS, React, TypeScript, Vite
- AI engineering: embeddings, FAISS, grounded RAG, evaluation
- DevOps: Docker, Docker Compose, AWS EC2, Nginx
- Documentation: architecture notes, demo scripts, submission evidence
