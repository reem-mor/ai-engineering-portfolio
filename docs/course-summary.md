# Course Summary

This repository documents hands-on work from the **Amdocs AI-Augmented Software Engineering** training program. It is organized as a portfolio: lectures build skills, homework applies them, and projects demonstrate end-to-end delivery.

Full curriculum map: [`docs/SYLLABUS.md`](SYLLABUS.md)

## Program flow

| Stage | Location | Focus |
|-------|----------|-------|
| Lectures 01–03 | [`lectures/`](../lectures/) | Python, Jupyter, OOP, NumPy |
| Lectures 04–06 | [`lectures/04_nlp_rag/`](../lectures/04_nlp_rag/) … [`06_flask_advanced_rag/`](../lectures/06_flask_advanced_rag/) | NLP, embeddings, FAISS, Flask, SQLite RAG |
| Lectures 07–11 | [`lectures/07_docker_aws/`](../lectures/07_docker_aws/) … [`11_local_models_webui/`](../lectures/11_local_models_webui/) | Docker, AWS, MCP, Bedrock, n8n, LangGraph, Open WebUI |
| Homework 01–03 | [`homework/hw01/`](../homework/hw01/) … [`hw03/`](../homework/hw03/) | Notebooks, Python exercises, CLI + tests |
| Homework 04 | [`homework/hw04/`](../homework/hw04/) | RAG app scaffold — full implementation in capstone |
| Homework 05–07 | [`homework/hw05/`](../homework/hw05/) … [`hw07/`](../homework/hw07/) | EC2/Docker/Nginx, n8n agent, Open WebUI + tools |
| In-repo projects | [`projects/`](../projects/) | Capstone RAG (IncidentIQ) + Bedrock iteration |
| External flagships | [`flagships/`](../flagships/) | PITER AiOps, HINDSIGHT, course-assistant-bot |

## Skills demonstrated

- Python 3.12+, validation, testing with pytest
- Web backends: Flask and FastAPI
- Frontend: HTML/CSS/JS, React, TypeScript, Vite
- AI engineering: embeddings, FAISS, grounded RAG, evaluation, Bedrock KB/Agent
- DevOps/SRE: Docker, Docker Compose, AWS EC2, Nginx, CI, health checks
- Agents: MCP, n8n, LangChain/LangGraph, Open WebUI tool servers
- Documentation: architecture notes, demo scripts, submission evidence
