# Oz Veruach Bot

Bilingual (HE/EN) **Telegram assistant** for course cohort ops — async Python, SQLAlchemy +
Alembic, multi-LLM routing, strict typing, Docker.

Built by [Re'em Mor](https://github.com/reem-mor).

## Quick start

```bash
uv sync
cp .env.example .env   # fill tokens locally — never commit
uv run oz-bot
uv run pytest
```
