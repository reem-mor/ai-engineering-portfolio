# amdocs-ai-course

Course workspace for the Amdocs AI-Augmented Software Engineering program.

## Structure

- lectures
- homework
- projects
- notes
- resources

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Workflow

Start:
```bash
git pull
```

End:
```bash
git add .
git commit -m "update"
git push
```
