# nginx-docker-lab

AWS EC2 + Docker + Nginx homework submission structure.

## Purpose

This repository is organized for the Ubuntu EC2, Docker Installation, and Nginx Container lab.
It keeps submission evidence and notes in one clean place.

## Structure

- `screenshots/` - proof of the EC2 instance, Docker/Nginx runtime, browser validation, and cleanup.
- `notes/submission_notes.md` - short checklist for what to submit by email.

## Required screenshots

1. `ec2-running.png` - AWS EC2 console showing the instance in Running state.
2. `docker-nginx-running.png` - SSH terminal showing Docker installed and the Nginx container running.
3. `nginx-browser.png` - Browser showing the Nginx welcome page from the EC2 public IP.
4. `ec2-terminated.png` - AWS console showing the instance termination step or terminated state.

## Suggested email content

- Attach the screenshots.
- Paste the GitHub repository link in the email body.
- Mention the lab name clearly in the subject line.

## Notes

Do not upload the `.pem` key file to GitHub.
Terminate the EC2 instance after finishing the lab to avoid unnecessary charges.
