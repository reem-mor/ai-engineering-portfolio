# HW07 Screenshots

Submission evidence generated by Playwright ([`e2e/submission-screenshots.spec.ts`](../e2e/submission-screenshots.spec.ts)).

## Regenerate

```powershell
cd homework\hw07\scripts
.\start-stack.ps1 -MockRapidApi
cd ..\e2e
npm install
npx playwright install chromium
npx playwright test submission-screenshots.spec.ts
```

Prerequisites: Ollama with a chat model, Docker Desktop, tool server on `:5005`.
